# e-learning-skillsprint

SkillSprint is a modern, responsive e-learning website designed to provide high-quality online courses with an engaging user experience.  
This project is built using HTML, CSS, and JavaScript.
## Features
- Responsive modern UI
- Hero section with call-to-action
- Courses showcase
- About and Features section
- Testimonials slider
- Contact form
- Smooth animations using AOS
- Font Awesome icons
- Google Fonts (Poppins)

## Technologies Used
- HTML5
- CSS3
- JavaScript
- AOS (Animate On Scroll)
- Font Awesome
- Google Fonts
